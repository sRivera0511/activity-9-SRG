# Activity 9 - Web Application Design

---

## Description
This project is a web application developed with Laravel 12 that implements a basic authentication system created manually (without using Breeze or UI packages). The application provides a landing page for guest users and a protected dashboard that can only be accessed by authenticated users.

---

## Features
- Landing page for guest users
- User registration system
- Login and logout functionality
- Protected dashboard using authentication middleware
- Dynamic navigation bar based on user session
- Access restriction for unauthorized users
